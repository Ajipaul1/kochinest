# KochiNest - Housing and Services Platform

![KochiNest Logo](https://via.placeholder.com/100x100?text=KN)

**KochiNest** is a comprehensive housing and services platform created for Kochi Corporation. This website provides a professional, authoritative solution for finding apartments and houses in Kochi, along with a complete suite of home services.

## 🌟 Features

### 🏠 **Housing Listings**
- **Free Listings**: Post property listings completely free of charge
- **Verified Listings**: All listings verified by Kochi Corporation for safety and security
- **Advanced Search**: Filter by property type, budget range, and location
- **Featured Properties**: Showcase premium listings with detailed information

### 🔧 **Complete Services**
- **Repair Services**: Professional home and appliance repair
- **Deep Cleaning**: Thorough cleaning for homes and offices
- **Security Services**: Property and event security solutions
- **Car Wash**: Premium car washing and detailing at your doorstep
- **Flat Shifting**: Professional moving and relocation services
- **Maintenance**: Regular property maintenance services

### 🌍 **Multilingual Support**
- **English** (Default)
- **Malayalam** (മലയാളം)
- **Hindi** (हिन्दी)

### 🔐 **User Authentication**
- **Login/Signup System**: Secure user registration and authentication
- **Form Validation**: Comprehensive client-side validation
- **Modal Management**: Smooth modal transitions and interactions

### 📱 **Responsive Design**
- **Mobile-First**: Optimized for all screen sizes
- **Accessibility**: WCAG-compliant design with proper focus states
- **Performance**: Fast loading and smooth animations

## 🛠️ Technology Stack

### Frontend Technologies
- **HTML5**: Semantic markup for better SEO and accessibility
- **CSS3**: Modern styling with CSS Grid, Flexbox, and custom properties
- **JavaScript (ES6+)**: Vanilla JavaScript with modern features
- **Font Awesome**: Icon library for consistent visual elements
- **Google Fonts**: Professional typography with Inter and Roboto

### Key Features
- **CSS Variables**: Consistent theming and easy customization
- **CSS Grid & Flexbox**: Modern layout techniques
- **Smooth Animations**: CSS transitions and keyframe animations
- **Form Validation**: Client-side validation with real-time feedback
- **Modal System**: Custom modal management with smooth transitions
- **Language Switching**: Real-time language switching with localStorage persistence
